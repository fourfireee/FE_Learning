# 背景

- 我之前的开发背景：计算机图形学、native 渲染引擎及工具、以及运行在 native 引擎之上的 shader 特效。
- 现在我要做的项目：
    - 01、基于一个自研 native 渲染引擎，开发一个前端 web app 作为特效工具。
    - 02、前端 AIGC workflow 编辑工具，提供类似 ComfyUI 的可视化 workflow 编排能力，调度自研和三方模型服务，组合出复杂 AIGC 效果。
- 为此我需要学习和掌握前端开发知识、基本概念，以及分析问题和解决问题的思考方式。只有掌握这些知识和思考方式，我再用 AI coding 辅助开发项目时，AI 在我手上才是可控的，我才具备判别 AI 生成代码是否是好代码的判断力，才能让项目可控。

# 写作和示例要求

- 用尽量通俗易懂的语言来精简地描述概念和计算原理。
- 必要的时候提供 md 文档内的 code snippet 示例或者 mermaid 流程图。
- 必要的时候提供独立的示例代码 folder。
- 代码里的关键点，需要在代码里增加注释，用同样的行文风格进行说明。
- md 文件的排版尽量简洁，除了标题，就用 bullet points，不要随意加粗或者斜体。
- 下面每个主要知识点都作为一个独立 md 文件，并在这个 `知识.md` 中引用。

# 学习目标

- 理解前端领域的基础逻辑和流程：`HTML -> DOM -> CSSOM -> Layout -> Paint -> Composite -> 事件响应 -> 页面更新` 全流程是如何发生的，中间的计算逻辑和渲染逻辑是什么。
- 理解前端技术栈的基本脉络和主要原理：HTML、CSS、JavaScript、TypeScript、React、Vue 分别解决什么问题，它们之间是什么关系。
- 理解浏览器与网络基础：HTTP / HTTPS、请求响应、缓存、跨域、Cookie / Token、浏览器存储、文件上传下载。
- 理解前端工程化基础：npm、包管理、模块化、Vite / Webpack、构建、环境变量、部署、Source Map。
- 掌握 React 相关的核心开发方式：组件拆分、props、state、hooks、副作用、组件通信、状态管理基础。
- 掌握分析问题和解决问题的思考方式：遇到问题时如何定位和解决，如何使用 DevTools、Network、Performance、Memory、React DevTools 等工具。
- 掌握面向可视化编辑器的前端能力：拖拽、缩放平移、节点连线、坐标系、命中测试、选择框、多选、撤销重做、快捷键。

# 运行方式

- 最简单方式：
    - 直接用浏览器打开对应示例目录里的 `index.html`。
- 更接近真实前端开发的方式：
    - 在仓库根目录启动一个静态服务器。

```bash
python3 -m http.server 5173
```

- 然后在浏览器访问：
    - `http://localhost:5173/examples/01-html-form-and-assets/`
    - `http://localhost:5173/examples/05-canvas-webgl-grayscale/`
    - `http://localhost:5173/examples/07-visual-editor-state/`
